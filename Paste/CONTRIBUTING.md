# Instrukcje Przyczyniania się (Contributing)

Dziękujemy zainteresowaniu przyczynieniem się do projektu Paste!

## Jak mogę pomóc?

- Raportowanie błędów
- Sugerowanie nowych funkcji
- Улучшение dokumentacji
- Wysyłanie pull requestów

## Proces

1. **Fork** repozytorium
2. Stwórz nowy branch: `git checkout -b feature/nazwa-funkcji`
3. Wyślij commity: `git commit -am 'Dodaj nową funkcję'`
4. Push do brancha: `git push origin feature/nazwa-funkcji`
5. Wyślij Pull Request

## Konwencje kodowania

- Używaj Swift style guide (Apple)
- Dodaj komentarze dla skomplikowanego kodu
- Używaj sensownych nazw zmiennych i funkcji
- Formatuj kod (Cmd+A, Ctrl+I w Xcode)

## Commit Messages

Pisz jasne i zwięzłe wiadomości commitów:

```
[FEATURE] Dodaj nową funkcję X
[FIX] Napraw błąd Y
[DOCS] Aktualizuj dokumentację Z
[REFACTOR] Refaktoryzuj moduł W
```

## Licencja
